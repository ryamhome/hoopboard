# HoopBoard Court Engine v2

本版重點：

- 新增 `js/court-engine.js`：真實比例球場規格引擎。
- 可切換 FIBA / NBA / NCAA / 半場訓練。
- 可切換經典木地板、NBA暗色風、紫金風、藍金風、街頭水泥。
- 左上角會顯示場地切換面板。
- 「檢查」按鈕會顯示目前場地規格。

## 上傳方式

解壓縮後，進入資料夾內，將以下內容全部拖到 GitHub Repo 根目錄：

- `index.html`
- `style.css`
- `js/`
- `backup/`
- `README.md`
- `OPTIMIZATION_ROADMAP.md`

不要只上傳 zip。
